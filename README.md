# Prettify JSON

Скрипт для отображения файлов формата .json в удобочитаемом виде.

# Quickstart

Example of script launch on Windows, Python 3.5:

C:\Users\Adderall\PretyPrintJson> python ppjson.py
Enter the path to the file: data.json
{'a': 123,
 'b': '4 '
      '5 '
      '6'}
