# Prettify JSON

Скрипт для отображения файлов формата .json в удобочитаемом виде.

# Quickstart

Example of script launch on Windows, Python 3.5:

Вызов скрипта: 
<code>python ppjson.py</code>

Исполнение скрипта:
Enter the path to the file: yourfile.json

{'a': 123,
 'b': '4 '
      '5 '
      '6'}
